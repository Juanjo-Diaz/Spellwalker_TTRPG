Según hemos visto que la base de datos que vamos a usar es MariaDB, es una base de datos con la que estamos bastante familiarizados y que cuenta con una buena compatibilidad con la sintaxis de Oracle, que nos facilitará hacer las consultas.

Guia de instalación:

Windows:  
1 \- Instalar XAMPP  
[https://www.apachefriends.org/download.html](https://www.apachefriends.org/download.html)

2 \- Completar los pasos de la instalación, aceptando las licencias de uso y las políticas

3 \- Una vez tengas instalado el XAMPP, activa la opción de Mysql y Apache

Linux:  
1- Instalarlo desde este enlace:   
[https://www.apachefriends.org/download.html](https://www.apachefriends.org/download.html)

Anotaciones importantes:

- Firewall: Asegúrate que tu Firewall está configurado y permite conexiones a MariaDB en el puerto 3306 si necesitas acceso remoto.  
- Contraseña de root: Es importante tener una contraseña fuerte en root para la instalación.	  
- Configuración avanzada: Para los entornos de producción necesitas hacer un ajuste de la configuración en los ficheros de MariaDB
